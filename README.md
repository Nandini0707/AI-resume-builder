# AI Resume Builder – MERN Stack

An AI-powered full-stack Resume Builder web application that helps users create, edit, preview, and download professional resumes with ease.

This project is built using the **MERN Stack** and integrates **AI features** to assist users in generating resume content.

---

## 🚀 Features

- User authentication (Login / Register)
- Create, edit, and manage multiple resumes
- Live resume preview
- AI-assisted content generation
- Resume download in PDF format
- Clean and responsive UI
- Secure backend APIs

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Redux Toolkit

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### AI Integration
- Google Gemini API (or other AI service)
